Core Capabilities

Everything the pipeline needs.
Nothing it doesn't.

⚡
Auto-Detection

Identifies your project type automatically — Flutter, React Native, Node.js, Python, Spring Boot, and more. No config required.

📱
Device Intelligence

Install to Android and iOS devices over USB, WiFi ADB, or QR code. Lists connected devices with model, version, and battery info.

🔐
Signing Pipeline

Generates production keystores, signs APKs, and verifies signatures. Full end-to-end signing in one command.

🚀
Deploy Grid

Deploy to Render, Railway, Vercel, Heroku, Docker registries, or GitHub Pages — all with a single `abp deploy` command.

🔌
Plugin System

Extend with bundled plugins: QR installer, Gradle optimizer, npm security auditor. Build and install your own.

📡
Termux Native

Built to run on Android via Termux — no root required. The only professional build system that lives on your phone.

🐳
Docker Ready

Production multi-stage Dockerfiles for every stack. Non-root runtime, health checks, Alpine/slim bases included.

☸️
Kubernetes

Production-ready K8s manifests with Deployment, Service, Ingress, resource limits, and liveness probes. One apply away.

Get Started

One command to install.

# One-liner install
curl -fsSL https://raw.githubusercontent.com/DNYFTETCH/DNYFappbuilder/main/install.sh | bash

# Verify
abp doctor
# Setup storage (first time only)
termux-setup-storage

# Install dependencies
pkg install git curl

# Install DNYFappbuilder
curl -fsSL https://raw.githubusercontent.com/DNYFTETCH/DNYFappbuilder/main/install.sh | bash

# Verify
abp doctor
git clone https://github.com/DNYFTETCH/DNYFappbuilder.git ~/dnyf-appbuilder
cd ~/dnyf-appbuilder && ./setup.sh
source ~/.bashrc
abp doctor
